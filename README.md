# Expression-Calculator
  管院信管专业乔昱嘉的代码，完成的是表达式计算器的基础题部分
## 项目简介
  这是一个使用 C++ 编写的数学表达式计算器，核心采用调度场算法（Shunting Yard Algorithm）进行语法解析和求值。它能够处理包含基本运算符、括号、一元负号及幂运算的复杂表达式，并正确遵循运算符优先级和结合性规则。
## 所支持的功能特性
* 基本四则运算：支持加 +、减 -、乘 *、除 /。
* 括号：支持圆括号 ( 和 ) 改变运算优先级。
* 负数与正数前缀：识别一元负号（如 -5、-(2+3)）和一元正号。
* 幂运算：支持 ^ 和 ** 两种形式，均表示乘方，且为右结合（例如 2^3^2 = 2^(3^2)）。
* 取模运算：支持 % 运算符，对浮点数使用 fmod。
* 优先级与结合性：严格遵循以下规则（优先级从高到低）：
```
   优先级        运算符         结合性

 1.  10       u-（一元负号）    右结合
 2.  9            ^ **         右结合
 3.  8           * / %         左结合
 4.  7            + -          左结合
```
* 错误提示：捕获并输出清晰的错误信息，包括：
1. 除零 / 取模零
2. 括号不匹配
3. 非法字符
4. 不完整的表达式
## 算法概述：调度场算法
  本实现采用双栈求值（操作数栈 + 运算符栈），无需显式转换为后缀。
### 核心流程
1. 词法分析：逐个字符扫描输入字符串，识别数字（整数/浮点数）、运算符（包括 **）、括号。
2. 处理数字：直接压入操作数栈。
3. 处理左括号：压入运算符栈。
4. 处理右括号：弹出运算符并执行计算，直到遇到左括号。
5. 处理运算符：
  * 若当前为 - 且期望一元运算符（un 标志为 true），则将其标记为 u-（一元负号）。
  * 否则为二元运算符。
  * 根据优先级和结合性决定是否弹出栈顶运算符进行计算：
      * 对于左结合运算符，当栈顶优先级 >= 当前优先级时弹出。
      * 对于右结合运算符（^ ** u-），当栈顶优先级 > 当前优先级时弹出。
  * 最后将当前运算符压栈。
6. 结束：将栈中剩余运算符全部弹出并计算，最终操作数栈顶即为结果。
### 一元负号的处理
   一元负号具有最高优先级，且为右结合。代码中通过 un 标志判断是否应将其解析为 u-（例如表达式开头或左括号之后）。计算时直接取操作数栈顶的相反数。
### 幂运算的右结合性
通过 prio 函数返回优先级，并在比较时对 ^ / ** 使用 cur > top 而非 cur <= top，确保右结合。

### 代码结构
```
calculator.cpp
├── prio(string op)      # 返回运算符优先级
├── calculate(a,b,op)    # 执行二元运算
├── is_op(char c)        # 判断是否为运算符字符
├── solve(string s)      # 核心求值函数（调度场实现）
└── main()               # REPL 循环
```
### 关键函数说明
* solve：接受表达式字符串，返回计算结果。内部维护两个栈 num（double）和 st（string），以及一个 un 标志（是否期望一元运算符）。
* prio：根据运算符字符串返回整数优先级，用于比较。
* calculate：根据运算符对两个操作数进行运算，并处理除零、取模零异常。
